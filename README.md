## Disabling Italics

To disable italics, add the following to your `settings.json`:

```json
"editor.tokenColorCustomizations": {
    "[Noir Nebula Theme]": {
        "textMateRules": [{
            "scope": [
                // Add your specific scopes here
            ],
            "settings": {
                "fontStyle": ""
            }
        }]
    }
}
```
